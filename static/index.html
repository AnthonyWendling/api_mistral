Connexions API

Connectez une table NocoDB (ou autre) pour indexer les documents enregistrés via les endpoints adaptés.

Chargement…

Ajouter une connexion NocoDB

Base URL de l'API NocoDB, token, ID de table. Collection : id fixe ou template ex. nocodb-affaire-{{affaire_id}}. Mapping : noms de champs NocoDB → paramètres (file_url, document_id, nocodb_record_id, affaire_id, numero_affaire).
